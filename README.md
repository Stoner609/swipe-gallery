## Overview

Swipe Gallery is a React-based swipeable gallery inspired by the carousel experience used on the early Swag website.

The project was developed by studying and adapting the original implementation, then refactoring it with React Hooks while exploring virtualized rendering techniques similar to react-window.

Rather than being a from-scratch implementation, this project focuses on understanding, modernizing, and improving an existing carousel architecture.

## Features

* Swipe-based card navigation
* React Hooks architecture
* Virtualized rendering for large datasets
* Smooth and mobile-friendly interactions
* Rendering performance optimization

## Motivation

The goal of this project was to explore how large collections of content can be presented through a swipe-based interface while maintaining good rendering performance.

By adapting an existing production-proven implementation and applying modern React patterns, the project serves as both a learning exercise and a frontend performance experiment.

## Tech Stack

* React
* React Hooks
* JavaScript (ES6+)
